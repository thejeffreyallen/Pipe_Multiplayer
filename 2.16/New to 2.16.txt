FrostyP Pipe Manager update

New to 2.16:

* Publicised server option
Any running server app's currently posting to the hub will show in game
Port forward still required for remote players to connect

* Snap to floor - Parkbuilder mode

* Replay fixes, speed markers



Existing users install:
1) Make sure you have TheGarage 2 installed.

2) Overwrite your /Mods/FrostyPManager folder with new folder.

or connect to a 2.16 server and follow instructions


Server setup additional steps:

* if you have multiple servers on the same IP, name them differently to have them show seperatly

* enter "y" when asked to publicise server.

* enter "n" for default API hub or "y" to specify POST,PUT addresses and Api key
