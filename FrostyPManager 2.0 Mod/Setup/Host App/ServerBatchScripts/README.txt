These files are only if you're wanting to run multiple servers at once. Each will auto restart if they crash. 
Currently they are set up to cover ports 7777-7780 MaxPlayers=15, tick=120. 
Edit the autoRestartServer.bat files with notepad++ or something to change these values.

If you wish to add more than 4 servers, copy and paste an existing one, then edit the 
values accordinlgy.

Run StartServers.cmd to start the servers
